### PL-02: shadcn Dialog ⚠️

**问题描述**：新建主题弹窗（Dialog）背景透明，背景内容穿透显示。

**截图**：
- `02-new-topic-dialog.png` - 可看到背景的"UT-44测试主题"等文字穿透到弹窗上
- `03-new-topic-form.png` - 表单字段可看到但无正确背景色

**分析**：
- Dialog 组件使用 `@base-ui/react` 实现
- 控制台有 Warning: `Function components cannot be given refs... Check the render method of DialogClose`
- 这是一个 React ref 传递问题，可能导致 Dialog 的样式层叠顺序异常

**建议**：检查 `components/ui/dialog.tsx` 中 DialogClose 的 forwardRef 使用
